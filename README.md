## 📖 About the Project

**LAU Ride** is a full-stack web application developed to simplify transportation for university students. The platform enables students to offer rides, request seats, manage bookings, and communicate securely through an intuitive interface.

---

## ✨ Features

- Secure user authentication
- Create and manage rides
- Search and request available rides
- Accept or reject passenger requests
- Student profile management
- QR code verification
- Emergency SOS feature
- Responsive user interface

---
